# Terittory
An ESX based TURF and TERRITORY script for FIVEM

# Features 
• KILLFEED    
• TIMER      
• Automatically sends announcemnt

# Commands
• /startzone - To Start zone       
• /stopzone - To Stop Zone        
• /turf - To send Announcement

# Dependencies 
  es_extended      
  ox_lib
